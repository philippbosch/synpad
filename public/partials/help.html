Creating a new document

You can either go to {{ dbname }}.synpad.net, and use the form to create a new document, or you can simply go to {{ dbname }}.synpad.net/name-of-your-document to choose a specific name. If that document does not exist yet, it will be created for you and you will be redirected to the edit view.

Editing an existing document

Say you want to edit the document foo which is located at {{ dbname }}.synpad.net/foo. Simply append an underscore _to the URL to go to the edit view for that document: {{ dbname }}.synpad.net/foo_.

If you currently looking at the read-only view of that document, you can also just hit the underscore _ key on your keyboard.

Publishing a document

Say you are editing the document foo at {{ dbname }}.synpad.net/foo_. By clicking the Publish button in the upper right corner, you make the current version of the document available at {{ dbname }}.synpad.net/foo.

Listing all documents

A list of all documents is available at {{ dbname }}.synpad.net/_list.
